# Resume Match Analyzer (C++ OOP Project)

## Overview
This project is a C++ based Resume Match Analyzer that compares a resume with a job description and calculates a match percentage. It also identifies missing keywords and provides suggestions for improvement.

## Features
- Resume and Job Description comparison
- Match percentage calculation
- Missing keyword detection
- Random improvement suggestions
- Implementation of core OOPS concepts

## OOPS Concepts Used
- Encapsulation
- Abstraction
- Inheritance
- Polymorphism

## Technologies Used
- C++
- File Handling
- STL (Vector, String, Algorithm)

## How It Works
1. User enters file paths for resume and job description.
2. Program reads both files.
3. Keywords from job description are extracted.
4. Resume is checked for these keywords.
5. Match score is calculated.
6. Missing keywords and suggestions are displayed.
